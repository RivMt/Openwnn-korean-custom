## 기능

하드웨어 키보드와 다양한 배열, 테마를 지원합니다.

### 배열

휴대전화와 컴퓨터에서 사용되는 다양한 배열을 지원합니다.

* 두벌식
* 세벌식
* 단모음
* 천지인
* 나랏글

### 테마

머티리얼 디자인에 기반한 3종과 전자책 리더기를 위한 E-Ink 테마가 제공됩니다.

* White
* Dark
* Flat Dark
* E-Ink
